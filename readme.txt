== Description ==

NCD Login/Logout plugin allows you connect,disconnect and register users to your website.
Highly customisable, it can be used as a widget.

== Installation ==
1. Upload `ncd_login_logout` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Plugin settings are located in 'NCD Plugins', 'NCD Login/Logout'.

= Technical support =


== Remove plugin ==

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress

It's best to use the build in delete function of wordpress. That way all the stored data will be removed and no orphaned data will stay.
